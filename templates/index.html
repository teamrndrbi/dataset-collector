📖 Panduan Setup

  1. Buka Google Cloud Console
  2. Enable Google Drive API
  3. Buat OAuth 2.0 Client ID (type: Web Application)
  4. Tambahkan redirect URI: http://localhost:5001/auth/callback
  5. Download Client Secret JSON
  6. Copy file ke folder app sebagai client_secret.json, atau paste isinya di field di atas
  7. Klik "Login Google Drive" → authorize di browser
  8. Buat folder di Google Drive → copy Folder ID dari URL
  9. Paste Folder ID di field "Parent Folder ID" → Simpan
